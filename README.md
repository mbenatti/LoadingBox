LoadingBox
==========

Simple Dart Loading Box


How to use:

```sh
void main() {
  //Create Box loading
  LoadingBox boxLoading = new LoadingBox();
  //Show Loading
  boxLoading.show();
  //hideLoading
  boxLoading.hide(); 
}
```
